# Restormel Documentation

Restormel is the platform and product family that SOPHIA showcases.

## Public clone note

**This repository does not ship the full Restormel planning pack** (strategy, architecture, delivery, design, and reference automation tree). Those materials are maintained separately for product and partner work.

Use this page for **orientation only**. For the live showcase application, start at [../sophia/README.md](../sophia/README.md).

## How to think about the split

- **SOPHIA** answers “what does the reference app do, how is it shaped, where is it going?”
- **Restormel** answers “what platform are we extracting toward, and in what order?” — with detail kept outside this public tree.
