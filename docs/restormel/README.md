# Restormel Documentation

Restormel is the platform and product family that SOPHIA now showcases.

This documentation set covers the platform strategy, modular architecture, product definitions, delivery sequencing, design and monetisation planning, plus the reference automation used to keep the planning pack operational.

SOPHIA remains the concrete app that proves these ideas in product form. Use [../sophia/README.md](../sophia/README.md) when the question is about the live showcase app. Use this section when the question is about platform meaning, extraction boundaries, roadmap structure, or delivery control.

## How to navigate this pack

- Start with the overview and strategy docs if you need the platform framing.
- Move into architecture and product docs when defining reusable boundaries or product surfaces.
- Use the delivery and design docs when sequencing work.
- Use the reference docs for automation, backlog seeding, and documentation maintenance rules.
