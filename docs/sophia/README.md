# SOPHIA Documentation

SOPHIA is the showcase and reference application for the Restormel platform.

It exists to prove the platform in a real product: graph-grounded reasoning, verification, retrieval explainability, BYOK and billing flows, API surfaces, and guided philosophy learning all meet here first before they are fully separated into platform modules.

## What SOPHIA is now

- a public-facing showcase app for graph-grounded philosophical reasoning
- a reference implementation for core Restormel platform capabilities
- a dogfooding environment where platform ideas are validated in product form
- the domain-specific experience layer, while Restormel remains the platform/product-family layer
