# SOPHIA: Structured Ontological & Philosophical Heuristic Intelligence Agent

A philosophical reasoning engine that applies three-pass dialectical analysis to questions, arguments, and policy decisions.

## Phases

- **Phase 1-2:** Three-pass engine validation + SvelteKit app (ethics domain)
- **Phase 3a-3f:** Knowledge base expansion (ethics → epistemology → political philosophy → philosophy of mind → philosophy of AI → remaining domains)
- **Phase 4:** Web search integration and gap-filling
- **Phase 5+:** Auth, commercial features, scaling

## Getting Started
```bash
npm install
npm run dev
```

See DEVELOPMENT.md for detailed setup.
